# CompatLab ArtifactDoctor 0.7

Release date: 2026-06-28

## Summary

CompatLab ArtifactDoctor 0.7 adds bundle-aware recursive dependency resolution.

The CLI can now inspect a shipped application directory, resolve local
`DT_NEEDED` shared libraries, follow transitive dependencies when requested, and
show whether each dependency is provided by the bundle, the selected target
profile, or neither.

## Added

- `--bundle-root DIR` for `scan` and `compare`.
- `--recursive` for transitive bundled ELF dependency traversal.
- Bundle indexing with conservative `--max-files` protection.
- Recursive traversal limit through `--max-depth`.
- `$ORIGIN` expansion for common `RUNPATH` and `RPATH` layouts.
- Local lookup through:
  - `RUNPATH`;
  - `RPATH`;
  - entrypoint/requester directory;
  - `lib`, `lib64`, `usr/lib`, and `usr/lib64` under the bundle root;
  - fallback bundle index by filename.
- Dependency resolution states:
  - `bundled`;
  - `target`;
  - `missing`;
  - `ambiguous`.
- Rich dependency resolution table in terminal reports.
- `dependency_graph` in JSON reports with nodes, edges, candidates, and
  unresolved dependencies.
- Compare mode now checks every recursively resolved bundled ELF against the
  selected target profile.
- Bundle resolver unit tests and CLI JSON coverage.

## Changed

- `compare` treats libraries resolved from the local bundle as available for
  compatibility checks, while still comparing those bundled libraries against
  the target profile.
- Existing single-artifact `scan` and `compare` behavior remains unchanged when
  `--bundle-root` is not provided.

## Current Behavior

Resolve a bundle recursively against a generated target profile:

```bash
uv run compatlab compare ./dist/my-app \
  --target-file /tmp/ubuntu-2204-cpp-runtime.yaml \
  --bundle-root ./dist \
  --recursive
```

Write a JSON report with dependency graph data:

```bash
uv run compatlab compare ./dist/my-app \
  --target ubuntu-2204 \
  --bundle-root ./dist \
  --recursive \
  --json report.json
```

Scan dependency resolution without a target profile:

```bash
uv run compatlab scan ./dist/my-app --bundle-root ./dist --recursive
```

## Not Included Yet

- Full dynamic linker emulation.
- Platform-specific loader cache behavior.
- Wheel, RPM, or DEB package scanning.
- SBOM generation.
- Vulnerability scanning.
- Automatic patching through `patchelf`.
- Running artifacts inside containers.
- HTML reports.
- Web interface, database, daemon, or server mode.
