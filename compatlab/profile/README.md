# Profile Package

Файл `compatlab.profile` структурирован по этапам рабочего процесса:

- `catalog.py`: загрузка встроенных и внешних файлов YAML с профилями целей.
- `builder.py`: создание объектов `TargetProfile` на основе обнаруженных `SystemFacts`.
- `local.py`: создание профиля ОС на текущем хосте.
- `docker.py`: анализ образов Docker, экспорт архивов rootfs в формате tar и создание профилей на их основе.
- `parsers.py`: анализ вывода команд хоста и извлечение метаданных.
- `rootfs.py`: анализ архивов rootfs Linux в формате tar.
- `runtimes.py`: предустановки пакетов среды выполнения для проверки контейнеров.
